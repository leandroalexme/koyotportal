## 1. Análise Cromática

### Verde EQI - Cor Institucional Principal

A paleta cromática da EQI é construída sobre tons de verde escuro institucional, transmitindo **crescimento, prosperidade e estabilidade** - valores essenciais para uma marca do setor financeiro.

| Token | Nome | HEX | RGB | Uso |
|-------|------|-----|-----|-----|
| `brand-primary` | Verde EQI Principal | `#004B3C` | `rgb(0, 75, 60)` | Backgrounds premium, logo |
| `brand-primary-light` | Verde Vibrante | `#00A67D` | `rgb(0, 166, 125)` | Links, destaques |
| `brand-accent` | Verde Destaque | `#2DC79C` | `rgb(45, 199, 156)` | CTAs, ícones, setas |
| `brand-surface` | Off-White | `#F0EDE9` | `rgb(240, 237, 233)` | Fundos claros |
| `brand-surface-dark` | Verde Profundo | `#003C30` | `rgb(0, 60, 48)` | Fundos escuros |

| `text-muted` | Texto Discreto | `#969696` | `rgb(150, 150, 150)` | Legendas, regulatório |

## 2. Tipografia

### Famílias Tipográficas

| Função | Fonte | Uso |
|--------|-------|-----|
| **Display** | Playfair Display | Headlines, títulos grandes, logo textual |
| **Heading** | Inter | Subtítulos, labels, UI |
| **Body** | Inter | Corpo de texto, parágrafos |

### Escala Tipográfica

#### Display (Playfair Display)
- **XL:** 72px - Hero headlines
- **LG:** 56px - Section headlines
- **MD:** 42px - Card titles
- **SM:** 32px - Subtitles

#### Heading (Inter)
- **H1:** 28px / Semibold
- **H2:** 24px / Semibold
- **H3:** 20px / Medium
- **H4:** 18px / Medium
- **H5:** 16px / Medium
- **H6:** 14px / Medium

#### Body (Inter)
- **LG:** 18px / Regular
- **MD:** 16px / Regular
- **SM:** 14px / Regular
- **XS:** 12px / Regular

#### Caption/Regulatório
- **MD:** 11px
- **SM:** 10px
- **XS:** 8px (textos legais/regulatórios)

### Parâmetros Tipográficos

| Propriedade | Headlines | Subtítulos | Corpo |
|-------------|-----------|------------|-------|
| Line Height | 1.1 | 1.4 | 1.6 |
| Letter Spacing | -0.02em | 0 | 0 |
| Caps Tracking | - | 0.1em | - |

## 5. Regras de Governança

### Elementos Travados (Locks)

| Elemento | Propriedades Bloqueadas | Editável Por |
|----------|------------------------|--------------|
| **Logo** | Todas (posição, tamanho, cor, conteúdo) | Apenas Owner |
| **Tagline** | Todas | Apenas Owner |
| **Títulos** | Fonte, tamanho, peso, cor, posição | Conteúdo por todos |
| **Campos de Dados** | Fonte, tamanho, formatação | Conteúdo por todos |
| **Imagens** | Posição, tamanho, bordas | Substituição permitida |

### Variáveis Semânticas

```typescript
// Cores
'brand-primary'      → #004B3C
'brand-primary-light'→ #00A67D
'brand-accent'       → #2DC79C
'brand-surface'      → #F0EDE9
'brand-surface-dark' → #003C30
'text-main'          → #003228
'text-light'         → #FFFFFF

// Fontes
'font-display'  → Playfair Display
'font-heading'  → Inter
'font-body'     → Inter
```

### Constraints de Valores

| Propriedade | Mín | Máx | Unidade |
|-------------|-----|-----|---------|
| Font Size (títulos) | 24 | 72 | px |
| Gap | 0 | 64 | px |
| Padding | 0 | 48 | px |
| Corner Radius | 0 | 24 | px |
| Opacity | 0.1 | 1 | - |
